# 🌸 10 AI Side Hustles — Interactive Roadmap

A beautifully designed, fully interactive HTML guide covering 
10 AI-powered side hustles for work-from-home moms — taking 
you from complete beginner to earning expert.

## What's inside

- 10 complete hustle roadmaps (Prompt Engineering → Data Labeling)
- 4-stage progression per hustle: Start → Build → Sell → Expert
- 40+ real, clickable resource links (free tools, courses, platforms)
- Animated income bars showing Starter / Growing / Expert earnings
- Real quotes from relatable people — not marketers
- Instagram Story reel format for easy social sharing
- Floral warm aesthetic — built for the WFH mom audience

## Tech stack

- Vanilla HTML, CSS, JavaScript — zero dependencies
- Google Fonts (Fraunces, Nunito, Caveat)
- IntersectionObserver for scroll animations
- Fully offline — open in any browser, no server needed

## How to use

1. Clone or download the repo
2. Open `ai_hustle_roadmap.html` in any browser
3. Use the sticky nav to jump between hustles
4. Click any resource link to go directly to the tool

## Who this is for

Work-from-home moms, career pivoters, and AI-curious beginners 
who want a clear, no-fluff path to earning real income using 
AI tools — starting with zero experience and zero budget.
